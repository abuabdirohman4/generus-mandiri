# Pengaturan

Halaman Pengaturan memungkinkan kamu mengelola akun, keamanan, dan preferensi aplikasi.

<YouTubeEmbed id="" title="Demo fitur Pengaturan" />

---

## Alur Penggunaan

<Steps>
  <Step title="Buka halaman Pengaturan">
    Klik ikon profil atau menu **Pengaturan** di sidebar. Halaman ini menampilkan daftar kategori pengaturan yang tersedia.

    <DocImage
      src="/images/docs/settings/01-halaman-pengaturan.png"
      alt="Halaman pengaturan"
      caption="Daftar kategori pengaturan"
    />
  </Step>

  <Step title="Ubah password">
    Klik kartu **Keamanan Akun** untuk membuka halaman ganti password. Masukkan password lama dan password baru, lalu klik Simpan.

    <DocImage
      src="/images/docs/settings/02-keamanan-akun.png"
      alt="Keamanan akun"
      caption="Form ubah password akun"
    />
  </Step>

  <Step title="Install aplikasi (PWA)">
    Klik kartu **Install Aplikasi** untuk panduan memasang aplikasi di perangkat kamu — tersedia untuk Android, iOS, dan desktop.

    <DocImage
      src="/images/docs/settings/03-install-aplikasi.png"
      alt="Install aplikasi"
      caption="Panduan install aplikasi di perangkat"
    />
  </Step>
</Steps>

---

## Tips

<Callout type="tip" title="Install di HP untuk akses lebih cepat">
  Install aplikasi sebagai PWA di HP kamu — bisa dibuka langsung dari layar utama tanpa perlu buka browser.
</Callout>

<Callout type="warning" title="Ganti password secara berkala">
  Untuk keamanan akun, disarankan mengganti password secara berkala dan tidak membagikan password kepada siapapun.
</Callout>
