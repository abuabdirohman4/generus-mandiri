# Selamat Datang di Generus Mandiri

Generus Mandiri adalah sistem manajemen sekolah untuk program pendidikan agama LDII.
Aplikasi ini membantu pengelolaan **presensi**, **laporan**, **data siswa**, dan **pengaturan akun** secara digital.

---

## Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| [Presensi](/docs/fitur/presensi) | Pencatatan kehadiran santri per pertemuan |
| [Laporan](/docs/fitur/laporan) | Rekap dan statistik kehadiran santri |
| [Siswa](/docs/fitur/siswa) | Data dan profil santri di kelompokmu |
| [Pengaturan](/docs/fitur/settings) | Kelola akun, keamanan, dan preferensi aplikasi |
