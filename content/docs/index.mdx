# Selamat Datang di Generus Mandiri

Generus Mandiri adalah sistem manajemen sekolah untuk program pendidikan agama LDII.
Aplikasi ini membantu pengelolaan **absensi**, **rapot**, **materi**, dan **organisasi** secara digital — dari tingkat kelompok hingga daerah.

---

## Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| [Absensi](/docs/fitur/absensi) | Pencatatan kehadiran siswa per pertemuan |
| [Rapot](/docs/fitur/rapot) | Pembuatan dan distribusi rapot digital |
| [Monitoring](/docs/fitur/monitoring) | Pantau progres dan kehadiran siswa |
| [Kelas](/docs/fitur/kelas) | Manajemen kelas dan penugasan guru |
| [Materi](/docs/fitur/materi) | Upload dan kelola materi belajar |
| [Organisasi](/docs/fitur/organisasi) | Struktur daerah, desa, dan kelompok |
| [Users](/docs/fitur/users) | Manajemen admin, guru, dan siswa |

## Panduan per Role

| Role | Akses |
|------|-------|
| [Superadmin](/docs/panduan/superadmin) | Akses penuh ke seluruh sistem lintas daerah |
| [Admin](/docs/panduan/admin) | Kelola data daerah / desa / kelompok |
| [Guru](/docs/panduan/guru) | Input absensi dan pantau perkembangan siswa |
